Search Console RegEx

Filter your GSC queries by intent, length, or keywords using custom RegEx.

Question Intent

Matches queries starting with What, How, Why, When, etc.

Branded Exclusion

Matches all queries EXCEPT those containing your brand name.

Long Tail (5+ Words)

Matches queries that have more than 5 words in them.

Transaccional Intent

Matches queries with Buy, Price, Cheap, Online.

^(how|what|why|when|where|who).*

Paste this into the GSC "Query" filter set to "Custom (Regex)".
